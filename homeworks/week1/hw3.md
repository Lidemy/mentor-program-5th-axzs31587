## 教你朋友 CLI
h0w 哥，首先你要先知道電腦有兩種方法可以操控，第一種叫做 Graphical User Interface（GUI, 圖形化使用者介面），簡單來說就是有個圖形化的介面讓你可以對電腦執行任何操作，像是我們在電腦裡面的檔案總管中可以新建資料夾，可以更改檔案的名稱那樣；而另一種方法就是你這次想知道的，叫做 Command Line Interface（CLI, 命令行介面），是透過文字的視窗進行操作，也就是利用純文字來操控電腦，需要靠輸入  Command Line 的指令來對電腦執行操作。
你可以到網路上搜尋 Git 然後進官網下載檔案安裝在你電腦上，安裝完後打開 Git Bash，就可以在這上面使用 Command Line 來操控電腦了，以下說明一下這次你會需要的指令：
1. mkdir 指令：新增資料夾
   用英文來看的話 Make Directory 就是新增一個資料夾的意思，所以輸入這個指令 + 空格 + 要新增的資料夾名稱，就可以新增一個資料夾了。
2. cd：切換資料夾
   用英文來看的話 Change Directory 就是換一個資料夾的意思，所以輸入這個指令 + 空格 + 要切換到的資料夾名稱，就可以切換到那個資料夾裡了。
3. touch 指令：建立檔案／更改檔案最後的修改時間
   touch 就是碰一下檔案的意思。當你叫電腦碰一下某個檔案，這個檔案最後的修改時間就會變成你剛剛叫電腦碰一下檔案的那個時間，所以最早的功用是拿來更改檔案最後的修改時間；不過當你要電腦碰一下的檔案不存在時，它就會幫我們建立這個檔案，所以也可以用來建立檔案，輸入這個指令 + 空格 + 要建立的檔案名稱，就可以建立檔案了。
